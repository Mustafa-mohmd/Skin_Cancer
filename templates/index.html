Skin Cancer Detection

Upload an image to get a disease prediction and see an enhanced view.

About This Project

This web application provides an interface for a skin cancer detection model. It allows you to upload a skin lesion image and returns a predicted class along with an enhanced version of the image to help visualize details more clearly.

  • Home page describes the project at a glance.
  • Next step takes you to the upload page.
  • Results show the predicted disease and the enhanced image.
